# socket.io-py-to-react
The Python request module passes the data to the React frontend as a POST request to the Socket.io server.

# Usage
**Socket.io & React On**
```bash
$ yarn install
$ yarn dev
```
**Send Data To Socket.io**
```bash
$ python socketio-py-to-react.py {title} {msg}
```
